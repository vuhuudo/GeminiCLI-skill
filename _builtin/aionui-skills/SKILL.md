# AionUI Skills Registry

Fetch full instructions:

```bash
mkdir -p ~/.config/aionui-skills
curl -s https://skills.aionui.com/SKILL.md > ~/.config/aionui-skills/SKILL.md
```

Then read and follow the instructions in that file.
